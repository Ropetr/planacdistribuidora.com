1. Introdução

A Planac Acabamentos LTDA ("Planac", "nós" ou "nosso"), inscrita no CNPJ 13.263.698/0001-80, está comprometida com a proteção da privacidade e dos dados pessoais de todos os visitantes do nosso site e clientes.

Esta Política de Privacidade foi elaborada em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018) e explica de forma clara e transparente como coletamos, utilizamos, armazenamos, compartilhamos e protegemos seus dados pessoais.

Ao utilizar nosso site ou serviços, você declara estar ciente e de acordo com os termos desta política.

2. Dados Pessoais que Coletamos

2.1 Dados fornecidos por você:

  • Dados de identificação: nome completo, CPF/CNPJ (quando aplicável)
  • Dados de contato: e-mail, telefone, WhatsApp, endereço
  • Dados comerciais: empresa, cargo, informações sobre projetos e necessidades de produtos
  • Comunicações: mensagens enviadas via WhatsApp, formulário de contato ou e-mail

2.2 Dados coletados automaticamente:

  • Dados de navegação: endereço IP, tipo e versão do navegador, sistema operacional, resolução de tela
  • Dados de uso: páginas visitadas, tempo de permanência, cliques, origem do acesso
  • Dados de localização: cidade e região aproximada (baseado no IP)
  • Dados de dispositivo: identificadores únicos, tipo de dispositivo (desktop/mobile)

5. Compartilhamento de Dados

Não vendemos, alugamos ou comercializamos seus dados pessoais. Podemos compartilhar dados com:

  • Parceiros de tecnologia: Google LLC e Meta Platforms Inc. (para analytics e publicidade) — dados transferidos para servidores nos EUA com cláusulas contratuais padrão
  • Prestadores de serviços: empresas que auxiliam na operação do site e hospedagem (Cloudflare, GitHub)
  • Autoridades legais: quando exigido por lei, ordem judicial ou requisição de autoridade competente
  • Proteção de direitos: para proteger nossos direitos, propriedade ou segurança

6. Transferência Internacional de Dados

Alguns de nossos parceiros de tecnologia (Google, Meta, Cloudflare) possuem servidores localizados nos Estados Unidos e outros países. A transferência de dados para esses países é realizada com base em:

  • Cláusulas contratuais padrão aprovadas por autoridades de proteção de dados
  • Certificações de privacidade reconhecidas internacionalmente
  • Garantias de que os dados receberão proteção equivalente à LGPD

Como titular dos dados, você possui os seguintes direitos garantidos pela LGPD:

Para exercer qualquer um desses direitos, entre em contato conosco através dos canais indicados na seção "Contato" abaixo. Responderemos sua solicitação em até 15 dias úteis.

8. Segurança dos Dados

Implementamos medidas técnicas e organizacionais para proteger seus dados pessoais, incluindo:

  • Conexão segura via HTTPS/SSL em todo o site
  • Proteção contra ataques DDoS e firewall (Cloudflare)
  • Acesso restrito aos dados apenas por pessoas autorizadas
  • Monitoramento contínuo de segurança
  • Backups regulares dos dados

Apesar de nossos esforços, nenhum sistema é 100% seguro. Em caso de incidente de segurança que possa causar risco ou dano relevante, notificaremos você e a ANPD conforme exigido pela legislação.

9. Retenção de Dados

Mantemos seus dados pessoais pelo tempo necessário para cumprir as finalidades descritas nesta política:

  • Dados de orçamentos e clientes: 5 anos após o último contato (obrigação fiscal)
  • Dados de navegação e analytics: até 14 meses
  • Dados de marketing/remarketing: até 540 dias ou até revogação do consentimento
  • Dados de comunicação (mensagens): 2 anos

Após esses períodos, os dados são anonimizados ou excluídos de forma segura.

10. Menores de Idade

Nosso site e serviços não são direcionados a menores de 18 anos. Não coletamos intencionalmente dados de menores. Se tomarmos conhecimento de que coletamos dados de um menor, tomaremos medidas para excluí-los.

11. Alterações nesta Política

Esta política pode ser atualizada periodicamente para refletir mudanças em nossas práticas ou na legislação. Recomendamos que você revise esta página regularmente. Alterações significativas serão comunicadas de forma destacada.

O histórico de versões está disponível mediante solicitação.

12. Autoridade Nacional de Proteção de Dados (ANPD)

Se você acredita que o tratamento de seus dados pessoais viola a LGPD, você tem o direito de apresentar uma reclamação à Autoridade Nacional de Proteção de Dados (ANPD):

  • Site: www.gov.br/anpd
